# 项目名--v1.3版本
这是一个基于ReAct框架的智慧教育Agent助手--小智。旨在提供孩子教育模式（引导带领孩子学习思考）和陪伴模式，还能提供家长模式（帮助家长解决基本问题并具备上网搜索基本功能）。

## 技术栈
ReAct
LangGraph  具体实现文档见docs/architecture/graph.md
RAG 具体实现文档见docs/architecture/rag.md
Memory 具体实现文档见docs/architecture/memory.md
Tools
